# Cargar datos desde CSV
import pandas as pd

archivo = filedialog.askopenfilename(
    filetypes=[("CSV", "*.csv")]
)
df = pd.read_csv(archivo)

# Exploracion basica
print(df.head())
print(df.describe())
print(df.isnull().sum())

# Limpieza: rellenar nulos con media
df.fillna(df.mean(), inplace=True)

Ventas Equilibradas

Las 6 lineas de producto muestran ventas similares. Food & beverages lidera con $56,144 USD, con apenas un 12% de diferencia respecto a la ultima categoria.

Metodos de Pago

Los tres metodos de pago (Ewallet, Cash, Tarjeta) tienen distribucion casi identica (~33% cada uno), indicando que los clientes no tienen preferencia fuerte.

~33% Distribucion equitativa

Rating Independiente
